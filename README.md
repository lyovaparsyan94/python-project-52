# Task Manager

## Prerequisites
- Python 3.10 or higher
- UV package manager

## Development

```bash
# Install dependencies
make install

# Run migrations
make migrate

# Format code
make format

# Check code
make lint

# Fix linting issues
make fix

# Run local server
make dev
